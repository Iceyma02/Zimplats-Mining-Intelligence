> **Nine years. 5.18M PGM ounces. $8.0B revenue. One intelligence layer.**

ZimplatsIQ is a data intelligence platform that transforms 9 years of audited mining reports (FY2017–FY2025) into actionable operational insights. It reveals patterns no single annual report shows — predictable safety deterioration following contractor surges, structural cost inflation masked by record PGM prices, and equipment failures costing $41M in lost revenue.

---

## 🎯 What This Project Does

| Problem | Solution |
|---------|----------|
| Annual reports hide multi-year patterns | 9-year unified dashboard |
| Safety only measured after injuries | Identified 6–12 month contractor → LTIFR lag |
| Equipment failures called "unexpected" | TMM downtime → $41M lost revenue quantified |
| Cost overruns routine but unexplained | 46% cost rise traced across 9 years |
| No predictive capability | $26–57M annual addressable value identified |

---

## 📊 Dashboard Modules

- **Overview** — 9-year revenue vs production, cost trend, anomaly detection
- **Production** — Milled tonnage (6.5Mt → 7.9Mt), grade model bias (8/9 years below forecast)
- **Financial** — Margin compression ($1,637/oz → $466/oz), $1.75B capital cycle
- **Safety** — LTIFR volatility, contractor correlation, 7 fatalities timeline
- **Environmental** — SO₂ index, water recycling (41% → 63%), solar transition
- **Workforce** — Headcount (5,943 → 9,150), revenue/productivity collapse (−48%)
- **Intelligence** — Cross-cutting findings, operational maturity score, addressable value

## 🔍 Key Intelligence Findings

| Finding | Impact |
|---------|--------|

## 🚀 Running Locally

```bash
# Clone the repository
git clone https://github.com/matechhub/zimplatsiq.git

# Navigate to directory
cd zimplatsiq

# Open with any web server
python -m http.server 8080
# or
npx serve .

# Open http://localhost:8080 in your browser
```

No build step. No installation. Open `index.html` directly or serve locally.

## 💡 Why This Matters

Zimplats spent **$1.75B on capital** across 9 years — upgraded Bimha, Third Concentrator, solar farm, Mupani ramp-up. The capital phase is nearly complete.

**What's next?** Operational efficiency.

But 9 years of data reveal **no predictive intelligence layer**:
- ❌ No early warning for safety deterioration
- ❌ No equipment condition monitoring
- ❌ No real-time grade reconciliation

This dashboard proves the value exists. The question is whether Zimplats builds the systems to capture it.
